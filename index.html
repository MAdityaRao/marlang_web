#include <stdio.h> int prime(int n) { if (n < 2) return 0; int i = 2; while (i * i <= n) { if (n % i == 0) return 0; i++; } return 1; } int main() { printf("Primes up to 20:\n"); for (int i = 2; i < 21; i++) { if (prime(i)) printf("%d\n", i); } return 0; } // gcc primes.c -o primes && ./primes

// Features

Built for
Engineers,
Not Scribblers.

Every decision in Mar was made to remove friction. You write logic. Mar handles the pipeline.

Format-Free I/O
Say goodbye to %d and &. print(a, b) and take(c) use C11 generic macros to safely auto-infer variable types at compile time.
print("Value:", x)
OOP with Classes
Define classes with fields and methods. Instantiate with new. Member access via clean dot syntax — no arrow operators.
Player p = new Player(100)
Smart Iteration
Python-style for i in range(a, b) and for item in arr loops. The C backend automatically infers array element types for you.
for item in arr
Clean Workspaces
mar run executes scripts invisibly. mar build creates native executables and instantly deletes intermediate C files.
mar run program.mar
C-Level Performance
Mar compiles to readable C code. You get GCC and Clang's full optimization pipelines at -O2 and beyond, for free.
.mar → .c → gcc → binary
Frictionless Syntax
No semicolons required. Declare multiple variables on a single line safely. Type casting uses clean function calls.
int a, b = 5, c
// How It Works

From Source
to Binary.

Mar is a classic multi-pass compiler. Every .mar file moves through four deterministic stages.

.mar
Source
your file
Clean Mar code with no semicolons and no boilerplate
[ ]
Lexer
lexer.c
Source text becomes a flat stream of typed tokens
/\
Parser
parser.c
Tokens become an Abstract Syntax Tree via recursive descent
{}
Codegen
codegen_c.c
AST is walked and emitted as readable, type-safe C code
cc
Compiler
gcc / clang
System C compiler applies full optimizations
OK
Binary
running
Native executable at full C speed
// Arena Memory Model

All AST nodes are allocated inside a single memory arena. One call frees everything at the end of compilation — no per-node free() required anywhere. Objects created with new and string concatenations in your Mar programs are also arena-allocated at runtime. Zero memory leaks by design.

// Install

Get Running
in 30 Seconds.

One-line install for macOS and Linux. Requires gcc or clang on your system.

Terminal
// macOS & Linux — one command
$curl -sSL https://raw.githubusercontent.com/MAdityaRao/mar-lang/main/install.sh | bash
// Verify the install
$mar --help
// Run your first program instantly
$mar run examples/hello.mar
Requirements: gcc or clang · make · git
macOS: xcode-select --install  ·  Linux: sudo apt install gcc make git
// Or build from source
$ git clone https://github.com/MAdityaRao/mar-lang.git
$ cd mar-lang
$ make
$ sudo cp bin/mar /usr/local/bin/mar
# Now use `mar` from any directory
