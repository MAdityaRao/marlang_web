// Features

Built for
Engineers,
Not Scribblers.

Every decision in Mar was made to remove friction. You write logic. Mar handles the pipeline.

No Semicolons
The parser uses statement structure and newlines to determine where expressions end. Focus on logic, not punctuation.
int x = 5
OOP with Classes
Define classes with fields and methods. Instantiate with new. Member access via clean dot syntax — no arrow operators.
Player p = new Player(100)
Range-Based For
Python-style range loops compile down to optimized C while loops. No index variables, no off-by-one accidents.
for i in range(0, 10)
One Command Run
Mar parses your source, generates C, invokes GCC or Clang, and executes your program. One step, zero config.
mar run program.mar
C-Level Performance
Mar compiles to readable C code. You get GCC and Clang's full optimization pipelines at -O2 and beyond, for free.
.mar → .c → gcc → binary
Clean Error Messages
Every error shows the exact line, column, and a caret pointing to the problem. Three levels: Lexer, Parser, Semantic.
7 | if(primme(i)) → ^
// How It Works

From Source
to Binary.

Mar is a classic multi-pass compiler. Every .mar file moves through four deterministic stages.

.mar
Source
your file
Clean Mar code with no semicolons and no boilerplate
[ ]
Lexer
lexer.c
Source text becomes a flat stream of typed tokens
/\
Parser
parser.c
Tokens become an Abstract Syntax Tree via recursive descent
{}
Codegen
codegen_c.c
AST is walked and emitted as readable C code
cc
Compiler
gcc / clang
System C compiler applies full optimizations
OK
Binary
running
Native executable at full C speed
// Arena Memory Model

All AST nodes are allocated inside a single memory arena. One call frees everything at the end of compilation — no per-node free() required anywhere. Objects created with new in your Mar programs are also arena-allocated at runtime. Zero leaks by design.

// Install

Get Running
in 30 Seconds.

One-line install for macOS and Linux. Requires gcc or clang on your system.

Terminal
// macOS & Linux — one command
$curl -sSL https://raw.githubusercontent.com/MAdityaRao/mar-lang/main/install.sh | bash
// Verify the install
$mar --help
// Run your first program
$mar run examples/hello.mar
Requirements: gcc or clang · make · git
macOS: xcode-select --install  ·  Linux: sudo apt install gcc make git
// Or build from source
$ git clone https://github.com/MAdityaRao/mar-lang.git
$ cd mar-lang
$ make
$ sudo cp bin/mar /usr/local/bin/mar
# Now use `mar` from any directory
